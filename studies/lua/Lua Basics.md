# Lua Basics

## Comments

```lua
-- this is a comment.

--[[ this is
a multiline comment
so many
--]]
```

## Variables Simple literals

```lua
local number = 5

local string = "hello, world"
local single = 'also works'
local crazy = [[ This
  is multi line and literal ]]

local truth, lies = true, false

local nothing = nil
```
