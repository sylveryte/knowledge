# Python Collections

- any [[Python Data Types|data types]] in list

```python
x = [4, True, 'hi']
```

- length `len(x)`, works with [[Python Strings]]
- append is **extend**
- **pop** removes last one and returns

```python
x.extend([4, 5, 6, 2, False])
a = x.pop()
```
