## Types

### Int

- Numbers as long as no `.`
- `-2323523`
- `83483839`

### Float

- Numbers with `.`
- `-2342342.0`
- `98238923.34`

### String

- Single or double quotation
- `'Hello'`
- `"Hello"`
- `'Heello"'` use one to embed other quotation
- [[Python Strings]]

### Bool

- `True`
- `False`

## Conversion

- `int('4')`
